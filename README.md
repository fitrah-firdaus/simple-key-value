# Dynamic Config Server 

This application build using GoFiber Framework. 

### Pre-requisite

- Golang 1.21
- MongoDB
- Redis

### Getting Started

- `git clone https://github.com/fitrah-firdaus/simple-key-value.git`
- `go get`
- `go run main.go`
- Open Web Browser pointing to `http://localhost:9090/health`
